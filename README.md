## jupyter notebook

* jupyter notebook 을 사용해 배운 알고리즘 실습하기!

* 설치하는 방법

  ```bash
  $ pip install notebook
  $ jupyter notebook
  ```

  jupyter notebook 을 사용하는 동안 주피터 노트북이 실행 중인 git bash를 종료하지 않도록 주의

* 종료하는 방법

  주피터 화면에서 우측상단 quit 클릭

  or 터미널에서 `ctrl + c`누르기
