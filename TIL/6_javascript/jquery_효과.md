# jquery 효과

* 기본 효과 메소드
  * .show()
  * .hide()
  * .toggle()
  * .slideDown()
  * .slideUp()
  * .slideToggle()
  * .fadeIn()
  * .fadeOut()
  * .fadeToggle()
* 구조
  * $(셀렉터).메서드(speed, callback);
    * speed: 효과 속도
    * callback: 효과 완료 후 실행 함수



##  animate()

$(셀렉터).animate(object, speed, easing, callback);



$(this)이벤트를 당하는 자기 본인을 뜻함
